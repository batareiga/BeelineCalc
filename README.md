# 📊 Калькулятор выгод Билайн

Интерактивный инструмент для подбора тарифов Билайн на основе потребностей клиента.

## 🚀 Быстрый старт

### Локальный запуск (разработка)

```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm run dev
```

Откройте http://localhost:5173 в браузере.

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы появятся в папке `dist/`.

---

## 📦 Деплой на бесплатные хостинги

### Netlify (рекомендуется — проще всего)

**Способ 1: Drag & Drop (без Git)**

1. Соберите проект: `npm run build`
2. Зайдите на [app.netlify.com/drop](https://app.netlify.com/drop)
3. Перетащите папку `dist` в окно браузера
4. Готово! Сайт опубликован

**Способ 2: Через GitHub (автоматический деплой)**

1. Создайте репозиторий на GitHub и загрузите файлы проекта
2. Зайдите на [app.netlify.com](https://app.netlify.com)
3. Нажмите **"Add new site"** → **"Import an existing project"**
4. Выберите GitHub → ваш репозиторий
5. Настройки сборки:
   - **Build command:** `npm run build`
   - **Publish directory:** `dist`
6. Нажмите **"Deploy site"**

---

### Vercel

1. Зайдите на [vercel.com](https://vercel.com)
2. Нажмите **"Add New Project"**
3. Импортируйте репозиторий GitHub
4. Vercel автоматически определит Vite-проект
5. Нажмите **"Deploy"**

---

### GitHub Pages

1. Установите `gh-pages`:
   ```bash
   npm install --save-dev gh-pages
   ```

2. Добавьте в `package.json`:
   ```json
   {
     "homepage": "https://ваш-username.github.io/имя-репозитория",
     "scripts": {
       "predeploy": "npm run build",
       "deploy": "gh-pages -d dist"
     }
   }
   ```

3. Задеплойте:
   ```bash
   npm run deploy
   ```

4. В настройках репозитория включите GitHub Pages для ветки `gh-pages`

├── index.html          # Точка входа

├── vite.config.js      # Конфигурация сборщика
├── src/

## ⚙️ Функционал

- **Параметры подбора:** интернет, минуты, бюджет
- **Профиль клиента:** MNP, AI-сервисы, Яндекс Плюс, кешбэк, комбо, детский тариф
- **Режимы отображения:** выгоды / характеристики
- **Генератор скриптов:** готовый текст для клиента
- **Обновление тарифов:** загрузка через AI (Claude API)

---

## 🔧 Обновление тарифов

В интерфейсе нажмите **"📁 Обновить тарифы"** → вставьте текст с сайта Билайн → AI автоматически извлечёт данные.

Или отредактируйте массив `TARIFFS` в `src/App.jsx` вручную.
